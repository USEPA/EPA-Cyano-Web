Gets the latest cyano concentrations for multiple locations of interest by uploading a CSV file of location coordinates.


  1. Create a CSV file containing location coordinates datatypes. The CSV must contain the following headers: latitude, longitude, and type, where "latitude" and "longitude" must be in decimal format within CONUS and "type" is either "daily" or "weekly" (Click to download an example CSV input file).
  2. Click the "Upload" button here in the "Run" tab and select the created CSV file. This will start the process of getting cyano concentrations for the locations in the uploaded CSV.
  3. Check the current job status and previous jobs in the "Jobs" tab.
  4. Once the job is complete, a spreadsheet of cyano concentrations for each location will be sent to your email.
