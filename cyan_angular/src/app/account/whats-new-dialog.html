What's new in v{{version}}

  • "Request Location Data" feature allows a user to upload a CSV of location coordinates which returns a CSV of cyano data for those locations.
  • "Waterbody Stats" feature provides pixel value counts of each waterbody geometry from the NHD HR waterbody dataset.
  • "Waterbody Reports" allow a user to download a report of waterbody (or multiple waterbodies) by county, tribe, or from the list of user's location.
