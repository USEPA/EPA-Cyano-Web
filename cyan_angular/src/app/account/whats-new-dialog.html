What's new in v{{version}}

  • Request Location Data feature allows you to request the data for multiple locations by uploading a CSV file containing the desired location coordinates.
  • Waterbody Stats feature provides waterbody aggregated cyano data and metrics for a collection of NHD HR waterbodies.
  • Waterbody Report features allows you to generate a waterbody data and metrics pdf report for a collection of waterbodies.
  • Waterbody Map Layer feature allows you to directly access and view the waterbody stats profile for a waterbody from the main map.
